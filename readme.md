# Flask Graph Example

## How to setup virtual environment

Setup **venv**.

```bash
python -m venv ./env
```

Install all dependencies.

```bash
pip install -r requirements.txt
```

## How to run

```bash
export FLASK_APP=app.py
flask run
```
